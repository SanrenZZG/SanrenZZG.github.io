5.自动类型转换

​ （1）数据类型按精度（容量）大小排序为：

​ char=>int=>long=>float=>double

​ byte=>short=>int=>long=>float=>double

​ （2）自动类型转换细节

​ 有多种类型的数据混合运算时，系统首先启动所有数据转换成容量最大的那种数据类型，然后再进行运算。

​ 当我们把精度（容量）大的数据类型赋值给精度（容量）小的数据类型时，就会报错，反之会自动转换。

​ byte、short、char 三者都能计算，在计算时首先转换为 int 类型，三者相互之间不能自动转换

​ boolean不参与转换。

​ 自动提升原则：表达式的结果的类型自动提升为，操作数中最大的类型。

6.基本数据类型的强制类型转换

自动类型转换的逆过程，将容量大的数据类型转换为容量小的数据类型。使用时要使用强制转换符 （）

短路与 && 与逻辑与 & 的区别：

​ 在==短路与==的判断过程中，如果第一个条件为==假==，则==不会继续检验第二个条件==，效率较高

​ 在==逻辑与==的判断过程中，如果第一个条件为==假==，则仍然判断第二个条件是否为真，效率较低

​ 在开发中多使用短路与，尽量不使用逻辑与

短路或 || 与逻辑或 | 的区别：

  • 在==短路或==的判断过程中，如果第一个条件为true，则==直接跳过第二个条件==，输出为true，效率较高
  • 在==逻辑或==的判断过程中，如果第一个条件为true，则==第二个条件也会执行==，效率较低
  • 开发过程中，大多会用短路或

八、标识符的命名的规则和规范

标识符概念：

java对各种变量、方法和类等命名时使用的字符序列称为标识符

标识符规则：

  • 由26个英文字母大小写，0~9，或 $ 组成
  • 数字不可以开头
  • 不可以使用关键字，但能包含关键字和保留字
  • java中严格区分大小写，长度没有限制
  • 标识符不能包含空格

标识符命名规范：

  • 包名：多单词组成时所有字母都小写：aaa.bbb.ccc
  • 类名、接口名：多单词组成时，所有单词的首字母大写：XxxYyyZzz【大驼峰】
  • 变量名、方法名：多单词组成时，第一个单词字母小写，第二个单词开始每个单词首字母大写：xxxYyyZzz【小驼峰】
  • 常量名：所有单词都大写，多单词时每个单词用下划线链接：XXX_YYY_ZZZ

数组

介绍：存放多个同一类型的数据，数组也是一种数据类型，属于引用数据类型

语法：基本数据类型 [] 数组名 表示对应数据类型的数组，

提示：可以通过 ==数组名.length== 得到数组的长度或大小

数组的动态初始化：

  • 先声明数组 语法：数据类型 数组名 [] 也可以 数据类型[] 数组名 eg：int a[];
  • 创建数组 语法：数组名 = new 数组类型[大小]； eg：a = new int[10];
    • 如果只声明数组，不创建数组，则在调用数组时会报错，此时的数组为null，会直接报错

数组的静态初始化：

​ 初始化数组：

​ 语法：数据类型 数组名[] = {元素值，元素值，元素值…..}

​ 说明：当数组元素已知且元素数量不多时，使用静态初始化；

使用注意事项：

  1. 数组是多个==相同类型==的数据的集合，（除非数据类型兼容，可以进行自动转换）实现对这对数据的统一管理
  2. 数组元素可以是任何元素，包括数据类型和引用类型，但是不能混用
  3. 数组创建之后，如果没有赋值，则会填充默认值 eg：String 为 null ；double 为 0.0 ；char 为 \u0000；boolean 为 false；其他为0
  4. 使用数组：（1）声明数组并开辟空间（2）给数组各个元素赋值（3）使用数组（4）下标从0开始
  5. 数组下标（索引）只能在指定范围内，否则会下标越界报错
  6. 数组属于引用类型，数组型数据是一个对象（object）

  1. 基本数据类型赋值：这个值就是具体的数据，而且相互不影响。
