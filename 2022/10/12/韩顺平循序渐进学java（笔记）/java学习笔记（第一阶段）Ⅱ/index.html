Switch分支结构

基本语法：

细节讨论：

  1. 表达式的数据类型，应该和 case 后的常量类型一致，或者可以自动转换成可以相互比较的类型 比如：输入的是字符，case 后常量类型为Int
  2. switch表达式中表达式的返回值必须是：（byte、short、int、char、enum[枚举]、String）中的一种
  1. case 子句中的的值只能为常量，不能为变量
  2. default 子句是可选的，当没有匹配的 case 时，执行 default ，如果没有default子句，也可以继续执行 switch 语句
  3. break 语句在执行完一个 case 分支后，跳出本次循环，如果没有 break 语句，则会顺序执行到 switch 语句结尾，除非遇到了 break 语句

练习1：使用switch把小写类型的char转换为大写（键盘输入）。只转换a，b，c，d，e。其他的输出other

练习2：对学生成绩大于60分的，输出“合格”，低于六十分的，输出“不合格”，（注：输入的成绩不能大于100）

练习3：根据用于指定月份，打印该月份所属的季节。3，4，5春季，6，7，8夏季，9，10，11秋季，12，1，2冬季[使用穿透]

Switch 语句和 if 的选择

  • 如果判断的数值不多，而且符合==byte、short、int、char、enum、String==这六种类型，虽然两个语句都可以使用，但推荐使用 switch 语句

  • 其他情况：对区间判断，对结果为 boolean 类型判断，使用 if，if 使用方面更广

For循环控制

基本语法：

  1. for关键字，表示循环控制。
  2. 四要素：（1）循环变量初始化（2）循环条件（3）循环操作（4）循环变量迭代

多重循环控制

介绍：

  1. 将一个循环放在另一个循环体内，就形成了嵌套循环，for、while、do..while均可以作为外层和内层循环（建议一般使用两层，最多不超过三层）
  2. 嵌套循环就是，内层循环作为外层循环的循环体。
  3. 外层循环次数为m次，内层为n次则内层循环体实际上要执行 m*n次

第六章 数组、排序、查找、多维数组

数组

介绍：存放多个同一类型的数据，数组也是一种数据类型，属于引用数据类型

语法：基本数据类型 [] 数组名 表示对应数据类型的数组，

​ 如：double[] test 定义一个变量名为 test 的double类型的数组

提示：可以通过 ==数组名.length== 得到数组的长度或大小

数组的动态初始化：

  • 先声明数组 语法：数据类型 数组名 [] 也可以 数据类型[] 数组名 eg：int a[];
  • 创建数组 语法：数组名 = new 数组类型[大小]； eg：a = new int[10];
    • 如果只声明数组，不创建数组，则在调用数组时会报错，此时的数组为null，会直接报错

数组的静态初始化：

​ 初始化数组：

​ 语法：数据类型 数组名[] = {元素值，元素值，元素值…..}

​ 说明：当数组元素已知且元素数量不多时，使用静态初始化；

使用注意事项：

  1. 数组是多个==相同类型==的数据的集合，（除非数据类型兼容，可以进行自动转换）实现对这对数据的统一管理
  2. 数组元素可以是任何元素，包括数据类型和引用类型，但是不能混用
  3. 数组创建之后，如果没有赋值，则会填充默认值 eg：String 为 null ；double 为 0.0 ；char 为 \u0000；boolean 为 false；其他为0
  4. 使用数组：（1）声明数组并开辟空间（2）给数组各个元素赋值（3）使用数组（4）下标从0开始
  5. 数组下标（索引）只能在指定范围内，否则会下标越界报错
  6. 数组属于引用类型，数组型数据是一个对象（object）

数组赋值机制

  1. 基本数据类型赋值：这个值就是具体的数据，而且相互不影响。

    eg：int n1 = 10; int n2 = n1; n2 = 80; 此时 n1=10；n2 的值不会影响 n1 的值，此时为值拷贝

  2. 数组赋值：在默认情况下是==引用传递==，赋的值为==地址==，赋值方式为==引用传递/地址拷贝==

    eg：int[] arr1 = {1 , 2 , 3}; int[] arr2 = arr1; 此时引用的是 arr1[] 的地址；

    当执行 arr2[0] = 10; 时；输出 arr1[] = {10 , 2 , 3}

排序

排序是讲一堆数据，以指定的顺序进行排序的过程。

排序的分类：

  1. 内部排序：指将需要处理的所有数据都加载到内部存储器中进行排序，包括（交换式排序法、选择排序法、插入式排序法等）；
  2. 外部排序：数据量过大，无法全部加载到内存中，需要借助外部存储进行排序。包括（合并排序法、直接合并排序法）
